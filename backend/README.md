Express + MongoDB backend for vehicle management system.

 Setup

1. Install dependencies:
powershell
npm install


2. Configure MongoDB:
   - Install MongoDB locally or use MongoDB Atlas
   - Update `.env` file with your MongoDB URI



Production mode
npm start
