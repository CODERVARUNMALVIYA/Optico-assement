# MERN Frontend (React + Tailwind)

## Production Deployment Configuration

Setup (Windows PowerShell):

```powershell
npm install
npm run dev
```

Open the URL shown by Vite (usually http://localhost:5173).

Notes:
- We added `tailwind.config.cjs` and `postcss.config.cjs` so Vite will process Tailwind.
- If you want a backend (Express + MongoDB) added, tell me and I can scaffold it next.
